# ArabSeed Clone — الصفحة الرئيسية (Home7)

نسخة كاملة من الواجهة الأمامية لصفحة `home7` من موقع عرب سيد.

## هيكل المشروع

```
arabseed-clone/
├── index.html          ← الصفحة الرئيسية
├── server.js           ← سيرفر Node.js بسيط
├── css/
│   ├── style.css       ← الستايلات الأساسية
│   ├── menu.css        ← قائمة الديسكتوب العلوية
│   └── drawer.css      ← القائمة الجانبية للموبايل + زر الهامبرغر
├── js/
│   └── script.js       ← جافاسكربت القائمة المتحركة
├── img/
│   └── logo.png        ← شعار الموقع
└── raw_source.html     ← السورس الأصلي كما هو (للمرجع)
```

## التشغيل المحلي

### الطريقة 1: Node.js (بدون أي مكتبات)

```bash
node server.js
```

ثم افتح المتصفح على: `http://localhost:3000`

### الطريقة 2: Python (إذا ما عندك Node)

```bash
python -m http.server 3000
```

### الطريقة 3: فتح الملف مباشرة

افتح `index.html` مباشرة في المتصفح (كلك يمين → Open with Browser).

## الاستضافة

### على استضافة عادية (Shared Hosting)
1. ارفع كل الملفات عبر FTP إلى مجلد `public_html`
2. الموقع بيشتغل تلقائياً

### على VPS
1. ثبّت Node.js
2. انسخ الملفات للسيرفر
3. شغّل `node server.js`
4. (اختياري) استخدم PM2: `pm2 start server.js`

### على Netlify / Vercel / GitHub Pages
1. ارفع المجلد كله
2. بيشتغل كموقع ستاتيك تلقائياً (ما تحتاج server.js)

## ملاحظات
- الموقع الأصلي مبني على WordPress لكن صفحة home7 هي صفحة HTML ثابتة بالكامل
- كل الـ CSS مدمج inline في الأصل، فصّلناه إلى ملفات منفصلة
- كل الروابط تم تحويلها إلى روابط نسبية (relative)
- السيرفر يخدم ملفات ستاتيك فقط بدون أي مكتبات خارجية
